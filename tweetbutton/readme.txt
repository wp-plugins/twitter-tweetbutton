 == Installation ==
1. Unpack the *.zip file and extract the /tweetbutton/ folder.

2. Drop the ‘tweetbutton’ folder into your ‘wp-content/plugins’ folder

3. In word press administration panels, click on plug-in from the menu.

4. You should see your new ‘TweetButton’ plug-in listed under Inactive plug-in tab.

5. To turn the word presses plug-in on, click activate.

6. Go to widget link under Appearance tab, Drag & drop TweetButton widget to your desired location in the active sidebar.

Parameters
Tweet Button Code : You have to enter your twitter username .

Choose your button : You can select your type of display like horizontal,vertical etc.

== Frequently Asked Questions ==

I installed the plugin but can’t see it. What’s wrong?
Make sure the directory name is entered correctly on the settings page.

Widget title appears but no widget?

Please check your twitter username .
